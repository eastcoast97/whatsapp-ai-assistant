# whatsapp-ai-assistant

Connect WhatsApp account via QR code to chat with an AI assistant through your own WhatsApp

## Tech Stack

- **Frontend**: react
- **Backend**: nodejs
- **Database**: postgresql
- **ORM**: prisma
- **Auth**: custom

## Getting Started

```bash
npm install
npm run dev
```
